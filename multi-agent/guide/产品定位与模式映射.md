# 产品定位与模式映射

## 定位

`multi-agent` 是可被其他 Agent / IDE / 脚本调用的**多模式协作运行时**：先选协作模式，再交付唯一结果。

## 方法论 → 产品模块

| 模式 | 方法论 | 产品模块 | CLI |
|------|--------|----------|-----|
| A 圆桌 | §2.2 | `modes.RoundtableRuntime` | `mode roundtable` |
| B Consult | §2.3 | `modes.ConsultRuntime` | `mode consult` |
| C Swarm | §2.4 STORM | `modes.SwarmRuntime` | `mode swarm` |
| 选型 | §2.5 | `selector.select_mode` | `run start --mode auto` |
| 轨迹 | §五 | `trajectory.TrajectoryStore` | `run trajectory` |
| 分片 | §六 | `context.ShardContext` | Swarm 内部 |
